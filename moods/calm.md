# 🌿 平静/治愈 (Calm)

## 情绪特征

**核心感受：** 平静、放松、安全、温暖、呼吸  
**典型场景：** 冥想、睡眠、学习、咖啡馆、瑜伽、自然漫步、心灵疗愈

---

## 编曲建议

| 维度 | 建议 |
|---|---|
| **调式** | 大调或小调均可，Dorian 调式有特别的平静感 |
| **BPM** | 50 – 90 |
| **节奏** | 稀疏、柔和，甚至无明显鼓点 |
| **和弦** | 缓慢和弦变化，七和弦、九和弦增添温柔感 |
| **音色** | 钢琴、原声吉他、弦乐 Pad、自然音效 |
| **人声** | 轻柔（可选），气声，不强制 |
| **制作** | 深混响，长延迟（Delay），动态轻柔 |

---

## 推荐风格

1. **Lo-fi Hip-Hop** — 学习/放松氛围，最流行的治愈风格
2. **Neo-Classical** — 钢琴或弦乐，纯器乐
3. **Ambient** — 无结构，纯氛围层次
4. **Indie Folk** — 吉他弹唱，温柔叙事
5. **Smooth Jazz** — 成熟、慵懒、轻松

---

## 场景细分

| 场景 | 推荐参数 | 关键音色 |
|---|---|---|
| 睡眠/冥想 | BPM 50–70，纯器乐 | 弦乐 Pad，自然音效，钢琴 |
| 学习/专注 | BPM 70–90，无人声 | Lo-fi 钢琴，鼓机，采样 |
| 咖啡馆 | BPM 80–100，可有轻柔人声 | 吉他，爵士鼓刷，钢琴 |
| 瑜伽 | BPM 60–80，流动感 | 长音符，呼吸感，无强拍 |

soft, gentle, atmospheric, [场景标签], lo-fi / classical / folk

```

piano melody, mellow drums, jazz samples, no lyrics, warm, cozy
```

```
ambient, meditative, peaceful, slow, 55 BPM, synthesizer pads,
nature sounds, no drums, deep reverb, healing, instrumental
```
