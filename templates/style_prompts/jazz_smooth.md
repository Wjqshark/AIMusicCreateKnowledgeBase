# StylePrompt 模版：Smooth Jazz (jazz_smooth)

## 适用场景
- 情绪：浪漫、放松、慵懒
- 场景：餐厅、咖啡馆、睡前、周末下午
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
smooth jazz, saxophone, piano, upright bass, brushed drums,
relaxing, [BPM] BPM, [情绪标签]
```

---

## 变体模版

### 夜晚爵士（经典）
```
smooth jazz, saxophone lead, piano comping, upright bass,
brushed drums, relaxing, evening mood, 95 BPM, lounge,
Kenny G inspired, no lyrics, warm
```

### Bossa Nova 爵士
```
bossa nova, acoustic guitar, female vocals in Portuguese or English,
jazz chords, 108 BPM, relaxing, Brazilian, sophisticated,
intimate, Astrud Gilberto inspired
```

### 现代 Nu-Jazz
```
nu-jazz, modern production, atmospheric, 90 BPM,
saxophone, electronic elements, chill, sophisticated,
no lyrics, Nils Frahm meets jazz
```

### 爵士人声
```
jazz vocal, female singer, standards, piano trio,
90 BPM, intimate, smoky bar atmosphere, Diana Krall inspired,
sophisticated, classic
```
