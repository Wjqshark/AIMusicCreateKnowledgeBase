# StylePrompt 模版：流行欢快 (Pop Upbeat)

## 适用场景
- 情绪：开心、欢快、积极
- 场景：夏日、恋爱、庆祝、青春
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
pop, upbeat, catchy melody, major key, bright production,
verse-chorus structure, radio-friendly, [BPM] BPM, feel-good
```

---

## 变体模版

### 夏日 Pop
```
summer pop, upbeat, fun, catchy chorus, acoustic guitar and synths,
female vocals, bright, 118 BPM, major key, beach vibes, euphoric
```

### Dance-Pop
```
dance pop, upbeat, 4/4 beat, driving bass, synth lead, female vocals,
catchy hook, 125 BPM, club-ready, energetic, layered production
```

### K-Pop 风格
```
K-pop, polished, upbeat, group vocals, synchronized energy, 120 BPM,
bright synths, punchy drums, catchy English/Korean hook, trendy production
```

### 青春励志
```
pop, inspirational, uplifting, anthemic chorus, piano and strings,
mixed vocals, 110 BPM, major key, triumphant, feel-good, empowering
```

---

## 调整指南

| 如果需要 | 修改/添加标签 |
|---|---|
| 更复古 | 加 `vintage`, `80s inspired`, `synthwave` |
| 更成熟 | 去掉 `fun`，加 `sophisticated`, `polished` |
| 男声 | 替换为 `male vocals` |
| 更商业 | 加 `radio-friendly`, `mainstream` |
| 更独立 | 加 `indie`, `lo-fi production`, `DIY feel` |
