# StylePrompt 模版：EDM Drop (edm_drop)

## 适用场景
- 情绪：激昂、派对、欢腾、释放
- 场景：派对、健身、节日
- 人声：可有可无
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
EDM, electronic, drop, build-up, 4/4, synthesizer, [BPM] BPM,
energetic, dance floor, [子风格]
```

---

## 变体模版

### Big Room House（节日派对）
```
big room house, EDM, festival, drop, 128 BPM, supersaws,
euphoric, crowd pleasing, no lyrics, Hardwell inspired,
build-up tension, powerful kick
```

### Future Bass（情感电音）
```
future bass, emotional, uplifting, 150 BPM, female vocal chops,
supersaws, heavy drop, melodic, bright synths, Flume inspired,
climactic, colorful
```

### Progressive House（深度）
```
progressive house, melodic, 125 BPM, emotional lead synth,
gradual build, hypnotic, no vocals, Eric Prydz inspired,
atmospheric, groove
```

### Dubstep（猛烈）
```
dubstep, aggressive, 140 BPM, wobble bass, heavy drop,
glitchy, intense, no lyrics, Skrillex inspired, industrial feel
```

### Deep House（夜店慵懒）
```
deep house, minimal, 122 BPM, groovy, late night, 
atmospheric, soulful vocals, warm bass, Chicago house feel
```
