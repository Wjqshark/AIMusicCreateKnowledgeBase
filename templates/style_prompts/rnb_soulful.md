# StylePrompt 模版：R&B 灵魂 (rnb_soulful)

## 适用场景
- 情绪：浪漫、深情、性感
- 风格：R&B, Neo-Soul, Soul
- 人声：女声为主
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
R&B, soulful female vocals, groove, rhodes piano, [BPM] BPM,
smooth, [情绪标签], minor key
```

---

## 变体模版

### Neo-Soul（文艺复古）
```
neo-soul, soulful female vocals, rhodes piano, jazz chords,
organic drums, warm, nostalgic, introspective, 85 BPM,
D'Angelo inspired, vintage, groove
```

### Contemporary R&B（现代流行）
```
contemporary R&B, female vocals, smooth, modern production,
trap hi-hats, 808 bass, emotional, 90 BPM, SZA inspired,
melodic, layered harmonies
```

### R&B Ballad（情歌）
```
R&B ballad, slow, romantic, female vocals, piano and strings,
soulful, 70 BPM, heartfelt, Beyoncé inspired, emotional climax
```

### Alt-R&B（另类）
```
alternative R&B, experimental, female vocals, atmospheric,
electronic elements, 88 BPM, moody, FKA twigs inspired,
dark, ethereal
```
