# StylePrompt 模版：浪漫流行 (pop_romantic)

## 适用场景
- 情绪：浪漫、甜蜜、温柔、爱意
- 场景：约会、情人节、婚礼、周年纪念
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
pop, romantic, love song, tender, piano, [BPM] BPM,
[人声], warm, heartfelt
```

---

## 变体模版

### 流行情歌（男声）
```
pop ballad, romantic, male vocals, piano and strings, 75 BPM,
tender, heartfelt, wedding song vibe, crescendo chorus,
emotional, John Legend inspired
```

### 流行情歌（女声）
```
pop, romantic, female vocals, upbeat yet tender, synth and piano,
105 BPM, sweet, summer love, catchy chorus, Taylor Swift inspired
```

### 对唱情歌
```
pop duet, romantic, male and female vocals, piano,
80 BPM, warm, sweet dialogue, love story, emotional,
Ed Sheeran style
```

### 婚礼主题
```
wedding song, romantic, orchestral pop, string quartet,
piano, 70 BPM, tender, cinematic, timeless, pure love,
vow ceremony, no rap
```
