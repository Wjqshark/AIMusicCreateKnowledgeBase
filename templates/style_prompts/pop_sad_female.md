# StylePrompt 模版：流行悲歌·女声 (Pop Sad Female)

## 适用场景
- 情绪：悲伤、失落、心碎、思念
- 场景：分手、失恋、离别
- 人声：女声为主
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
pop ballad, sad, emotional, female vocals, piano driven,
minor key, slow, heartbreak, [BPM] BPM, cinematic
```

---

## 变体模版

### 钢琴失恋情歌
```
pop ballad, heartbreak, emotional, female vocals, solo piano,
minor key, 70 BPM, raw, vulnerable, intimate, slow build to chorus
```

### 弦乐编配版
```
pop ballad, sad, soulful female vocals, piano and string orchestra,
minor key, 75 BPM, cinematic, sweeping, tearful, Adele inspired
```

### 带希望的悲歌（小调主歌 → 大调副歌）
```
pop, bittersweet, emotional journey, female vocals, piano,
minor verse, major chorus, 80 BPM, hopeful ending, resilient
```

### 现代制作悲歌
```
sad pop, alternative, female vocals, sparse production, minimal beats,
atmospheric synth, 85 BPM, melancholic, Billie Eilish inspired,
whispered verses, explosive chorus
```

---

## 调整指南

| 如果需要 | 修改/添加标签 |
|---|---|
| 男声版本 | 替换为 `male vocals`, `baritone` |
| 更古典 | 加 `no drums`, `classical strings` |
| 更现代 | 加 `electronic elements`, `808 bass` |
| 更治愈 | 从 `heartbreak` 改为 `healing`, `self-love` |
