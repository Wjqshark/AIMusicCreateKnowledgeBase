# StylePrompt 模版：Trap 嘻哈 (hiphop_trap)

## 适用场景
- 情绪：暗沉、力量、街头
- 场景：说唱、夜晚、炫酷
- 人声：男声说唱
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
trap, hip-hop, dark, 808 bass, hi-hat rolls, rap vocals,
[BPM] BPM, atmospheric, hard-hitting
```

---

## 变体模版

### Dark Trap
```
dark trap, ominous, male rap vocals, 808 bass, triplet hi-hats,
140 BPM, minor key, atmospheric piano, melodic hook,
Travis Scott inspired, cinematic
```

### Drill（芝加哥/英国）
```
UK drill, dark, aggressive, 140 BPM, sliding 808, hi-hats,
male rap vocals, gritty, street, British accent optional,
Pop Smoke inspired
```

### Melodic Trap（旋律 Rap）
```
melodic trap, emotional, male vocals singing and rapping,
140 BPM, atmospheric synths, 808 bass, sad, introspective,
Future inspired, autotune
```

### Lo-fi Hip-Hop（无人声）
```
lo-fi hip hop, chill, 82 BPM, jazz samples, mellow piano,
vinyl crackle, soft drums, no lyrics, study music, warm
```
