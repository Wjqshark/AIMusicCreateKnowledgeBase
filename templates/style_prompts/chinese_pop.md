# StylePrompt 模版：华语流行 (Chinese Pop / C-Pop)

## 适用场景
- 语言：中文歌词
- 风格：华语主流流行、C-Pop、国风
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
Chinese pop, Mandarin vocals, catchy melody, pop production,
verse-chorus structure, [BPM] BPM, [情绪标签]
```

---

## 变体模版

### 华语情歌（通用）
```
C-pop, Mandarin love song, emotional, female vocals, piano and strings,
minor key, 85 BPM, heartfelt, ballad, Asian pop production
```

### 国风流行（融合古典元素）
```
Chinese pop, guzheng, erhu, traditional Chinese instruments,
modern pop production, Mandarin lyrics, pentatonic melody,
80 BPM, cultural fusion, cinematic
```

### 华语摇滚
```
Chinese rock, Mandarin male vocals, electric guitar, powerful drums,
energetic, 130 BPM, 华语摇滚, anthemic chorus
```

### 华语 R&B
```
Chinese R&B, Mandarin vocals, soulful, groove, rhodes piano,
minor key, 90 BPM, smooth, Jay Chou influenced, sophisticated
```

### 古风/仙侠风格
```
ancient Chinese style, guqin, dizi flute, traditional percussion,
pentatonic scale, ethereal female vocals, cinematic, 75 BPM,
fantasy, wuxia inspired, orchestral fusion
```

### 华语民谣
```
Chinese folk, Mandarin, acoustic guitar, storytelling, warm,
simple arrangement, 85 BPM, 赵雷 inspired, nostalgic, heartfelt
```

---

## 重要提示

> Suno 和 Udio 对中文输入的支持有限，建议在 StylePrompt 中用英文描述风格，歌词输入部分再使用中文。
