# StylePrompt 模版：史诗交响 (Epic Orchestral)

## 适用场景
- 风格：电影配乐、游戏音乐、史诗背景音乐
- 情绪：激昂、壮阔、感人、史诗
- 人声：无歌词（可选合唱）
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
orchestral, epic, cinematic, full symphony orchestra, no lyrics,
instrumental, [情绪标签], [BPM] BPM, film score
```

---

## 变体模版

### 战争/史诗战斗
```
epic orchestral, battle music, powerful, brass heavy, percussion,
choir, intense, 130 BPM, cinematic, Hans Zimmer inspired,
dramatic, no lyrics, game soundtrack
```

### 情感/感人
```
orchestral, emotional, strings led, piano, swell, tearful,
cinematic, 75 BPM, no lyrics, Thomas Newman inspired,
gentle to powerful, dynamic contrast
```

### 冒险/奇幻
```
orchestral, adventure, fantasy, heroic, woodwinds and strings,
French horn, uplifting, 120 BPM, John Williams inspired,
quest theme, no lyrics, magical
```

### 黑暗/反派主题
```
dark orchestral, villain theme, ominous, low brass, dissonant,
choir chanting, minor key, 90 BPM, dramatic, tense, no lyrics
```

### 安静/冥想
```
chamber orchestra, minimal, strings quartet, peaceful, 60 BPM,
atmospheric, no percussion, meditative, no lyrics, Arvo Pärt inspired
```
