# StylePrompt 模版：民谣原声 (folk_acoustic)

## 适用场景
- 情绪：平静、治愈、思念
- 风格：Folk, Indie Folk, 中国民谣
- 人声：任意
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
folk, acoustic guitar, [子风格], fingerpicking, [BPM] BPM,
[情绪标签], storytelling, organic, natural
```

---

## 变体模版

### Indie Folk（文艺小众）
```
indie folk, acoustic guitar fingerpicking, male vocals, soft,
intimate, melancholic, 90 BPM, violin accents, Bon Iver inspired,
nature imagery, whispery
```

### 中国民谣
```
Chinese folk, acoustic guitar, Mandarin lyrics, storytelling,
warm, nostalgic, pentatonic melody, 85 BPM, simple arrangement,
赵雷 inspired, heartfelt
```

### Celtic Folk（凯尔特）
```
Celtic folk, fiddle, acoustic guitar, flute, uplifting,
traditional, 110 BPM, Irish jig feel, energetic, dance-like
```

### 治愈民谣
```
folk, healing, peaceful, acoustic guitar, soft female vocals,
70 BPM, gentle, minimal, warm reverb, nature sounds, tender
```

### Country Folk（乡村民谣）
```
country folk, acoustic guitar, steel guitar, male vocals,
75 BPM, storytelling, Americana, Southern warmth, honest
```
