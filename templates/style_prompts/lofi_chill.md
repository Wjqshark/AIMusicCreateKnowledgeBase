# StylePrompt 模版：Lo-fi 治愈 (Lo-fi Chill)

## 适用场景
- 情绪：平静、放松、专注
- 场景：学习、工作、冥想、咖啡馆
- 人声：无人声
- 平台：Suno v3/v4、Udio

---

## 基础模版

```
lo-fi hip hop, chill, relaxing, no lyrics, instrumental,
vinyl crackle, 80 BPM, mellow, study music
```

---

## 变体模版

### 学习专注
```
lo-fi hip hop, study music, chill, mellow, 82 BPM, piano melody,
jazz samples, soft drums, vinyl crackle, no lyrics, warm, cozy,
rainy day atmosphere
```

### 睡眠/冥想
```
lo-fi ambient, sleep music, very slow, 60 BPM, minimal, soft piano,
nature sounds, no drums, deep reverb, no lyrics, peaceful, dreamlike
```

### 咖啡馆午后
```
lo-fi jazz, chill, afternoon, 88 BPM, acoustic guitar, light drums,
café atmosphere, warm, no lyrics, bossa nova influence, organic
```

### 城市夜晚
```
lo-fi hip hop, night vibes, urban, melancholic, 78 BPM, piano,
muted trumpet, city sounds, rain, late night, no lyrics, atmospheric
```

## 调整指南

| 如果需要 | 修改标签 |

| 更专注 | 加 `focus music`, `alpha waves feel` |
| 更复古 | 加 `vintage`, `cassette tape feel` |
