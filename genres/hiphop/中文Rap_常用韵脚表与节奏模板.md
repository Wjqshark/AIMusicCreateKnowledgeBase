# 中文 Rap 常用韵脚表 + 节奏模板例句

> 用途：这份文档用于中文 Rap 写词速查。  
> 前半是常用韵脚表，后半是节奏模板例句。  
> 取向：偏普通话中文说唱实战，不是严格语言学教科书。

---

## 一、先记 5 条最重要的规则

1. 押韵先看韵母，不看字形。
2. 句尾韵脚最重要，句中内部押韵是加分项。
3. -n 和 -ng 不要默认混押。
4. zhi / chi / shi / ri 的 -i，不要直接等同于 ni / mi / li 的 i。
5. 双字押、短语押，比单字押更像成型的中文 Rap。

## 第一部分：中文 Rap 常用韵脚表

## 说明

- 下表按实战常用听感分组。
- 稳押表示更适合句尾主韵。
- 可近押表示可以尝试，但一定要自己念出来确认。
- 代表字只是示意，不是完整词库。

## 1. 开口韵 / 亮口韵

| 韵脚组 | 常见韵母 | 代表字 | 听感 | 建议 |
|---|---|---|---|---|
| a 组 | a, ia, ua | 花、下、话、炸、挂 | 开阔、直给、外放 | 很适合副歌、态度句 |
| o / uo 组 | o, uo | 破、火、过、说、落 | 圆、厚、口型明显 | 适合 hook、口号型 |
| e / ue / ue 组 | e, ie, ue | 夜、别、写、绝、月 | 稍紧、利落 | 适合情绪句、收束句 |
| ao 组 | ao, iao | 到、笑、炮、跳、照 | 明亮、外冲 | 很适合 punchline |
| ou / iu 组 | ou, iu(iou) | 后、走、丢、流、酒 | 滚动感强 | 适合轻快 flow |

## 2. 尖口韵 / 高位韵

| 韵脚组 | 常见韵母 | 代表字 | 听感 | 建议 |
|---|---|---|---|---|
| i 组（普通 i） | i, yi | 你、笔、戏、意、力 | 细、利、清楚 | 适合密集句、技术流 |
| u 组 | u, wu | 路、住、雾、出、数 | 闭口、稳重 | 适合压低声区 |
| u 组（v 系） | v, ve, vn | 绿、句、续、去、雨 | 冷、紧、锐 | 适合冷感 beat |

### 注意

下面这几类不要随便和普通 i 混成一组：

| 特殊组 | 代表字 | 说明 |
|---|---|---|
| zhi / chi / shi / ri 组 | 知、吃、事、日 | 这个 -i 的实际口感和普通 i 不一样 |
| zi / ci / si 组 | 子、次、四 | 这个 -i 也不要默认和你 / 里 / 记硬并 |

实战里可以偶尔近押，但不要把它当主韵默认混用。

## 3. 鼻音韵（最常用）

| 韵脚组 | 常见韵母 | 代表字 | 听感 | 建议 |
|---|---|---|---|---|
| an 组 | an, ian, uan, van | 看、慢、天、晚、远 | 平稳、口型清晰 | 很适合叙事型 verse |
| en 组 | en, in, un, vn | 门、真、心、云、分 | 紧密、细腻 | 适合中速推进 |
| ang 组 | ang, iang, uang | 光、响、场、强、晃 | 大、开、炸 | 适合大开大合句尾 |
| eng 组 | eng, ing, ueng | 城、等、冷、影、赢 | 紧、冷、硬 | 适合硬核、技术型 |
| ong / iong 组 | ong, iong | 风、空、梦、冲、凶 | 厚、卷、滚动 | 适合 trap 和 hook |

### 重点提醒

- an 和 ang：不要默认混押
- en 和 eng：不要默认混押
- in 和 ing：不要默认混押
- ong 和 eng：听起来接近时也别随便合并

可以偶尔做近押，但主韵最好分开。

## 4. 复合韵 / 滚动感强韵脚

| 韵脚组 | 常见韵母 | 代表字 | 听感 | 建议 |
|---|---|---|---|---|
| ai 组 | ai, uai | 来、海、白、快、在 | 亮、直接 | 很适合 hook 和记忆句 |
| ei / ui 组 | ei, ui(uei) | 飞、累、归、追、对 | 轻快、顺口 | 适合 flow 滚动 |
| ia / ua 组 | ia, ua | 压、下、挂、花、刷 | 开口大、好喊 | 适合现场感 |
| ie 组 | ie | 别、夜、裂、灭、街 | 短、利、急 | 适合加速段 |
| uai / ui 组 | uai, ui | 快、甩、坏、追、退 | 有滑行感 | 适合切分变化 |

## 5. 不建议轻易当主韵的组

| 韵脚组 | 代表字 | 原因 |
|---|---|---|
| er 组 | 而、二、耳、儿 | 独立性太强，不好大面积铺开 |
| 轻声虚词组 | 的、了、呢、吗、啊 | 句尾力量弱，容易塌 |
| 生僻儿化尾 | 这儿、那儿、门儿 | AI 和真人都容易含混 |

## 6. 写词最实用韵脚优先级

### 第一梯队：最稳
- ang
- ing / eng
- an
- ao
- ou
- ai
- o / uo

### 第二梯队：很好用但更挑口型
- en / in / un
- ong / iong
- ie
- ia / ua
- v 类

### 第三梯队：谨慎做主韵
- er
- 特殊 -i
- 轻声虚词尾

### 想写顺一点

## 第二部分：双字押 / 短语押示例

## 1. 单字押（入门）

- 灯光很亮 / 心事很烫
- 风吹过巷 / 脚步没晃

## 2. 双字押（推荐）

- 城市太宽阔 / 记忆太斑驳
- 说得很干脆 / 做得很干净

## 3. 短语押（更像成型 Rap）

- 我把夜色写成街头光亮
  你把沉默留在背后方向

- 我在旧楼之间对焦信号
  你在雾里缓慢退到尽角

## 第三部分：中文 Rap 节奏模板例句

## 使用说明

- / 表示一个小切分或停顿
- | 表示一小节结束
- 加粗代表建议重读或句尾韵脚
- 下面例句都是原创模板，用于感受节奏

## 模板 1：稳拍叙事（90-96 BPM）

### 节奏感觉
一拍一重心，句尾清楚，适合中文主歌入门。

### 例句
- 我站在 / 楼顶看 / 城市把 / 灯全点亮 |
- 风吹过 / 旧外套 / 心里那 / 句话没讲完 |

### 适合
- 现实题材
- 情绪表达
- 叙事型 verse

## 模板 2：一拍两字（基础二连音）

### 节奏感觉
每拍两个主要音节，中文常用推进型 flow。

### 例句
- 城市 / 灯火 / 压住 / 沉默 |
- 情绪 / 滚过 / 句尾 / 收着 ang |

### 适合
- 日常 verse
- 中速 beat
- 新手稳定练习

## 模板 3：二字组推进（最适合中文）

### 结构
2 + 2 + 2 + 2

### 例句
- 旧楼 / 霓虹 / 逆风 / 低空 |
- 这城 / 很重 / 但我 / 没怂 |

## 模板 4：2 + 2 + 3

### 结构
前面稳，最后一组加长度，句尾更像抛出去。

### 例句
- 别急 / 跟拍 / 把字放对上 |
- 旧梦 / 很亮 / 但人走得太快 |

## 模板 5：3 + 2 + 2

### 结构
开头更口语，后面收紧。

### 例句
- 我记得 / 那条 / old 街 |
- 你转身 / 太快 / 没停 |

## 模板 6：留白型 cadence

### 节奏感觉
故意空拍，不把拍子填满。

### 例句
- 我在这 / 看着你 / -- / 不急着回答 |
- 这段路 / 走很久 / -- / 还没到终点 |

## 模板 7：局部三连音（别全段都用）

### 节奏感觉
在一拍里平均塞 3 个短音，形成加速感。

### 口令
ONE-e-uh / TWO-e-uh / THREE-e-uh / FOUR-e-uh

### 例句
- 别着急 / 看着我 / 跟着走 / 别退后 |
- 把这段 / 压更紧 / 再推进 / 冲到顶 |

### 用法建议
- 前面 2 bars 稳拍
- 后面 1-2 bars 三连音加速
- 然后切回稳拍

## 模板 8：抢拍进入

### 节奏感觉
字提前半步进来，制造推力。

### 例句
- （提前）我早就 / 说过这 / 条路得 / 自己去闯 |
- （提前）别等到 / 全熄火 / 才想起 / 你还在扛 |

## 模板 9：后拍拖落

### 节奏感觉
句尾往后坐，更松、更有 swagger。

### 例句
- 我走得 / 很慢但 / 每一步 / 都算数...吧 |
- 你说得 / 很满可 / 真落地 / 还差点...火 |

## 模板 10：主韵 + 内部押韵

### 节奏感觉
句尾有主韵，句中埋一层内部韵。

### 例句
- 夜色在 / 楼侧外 / 折射开 / 还是太亮 |
- 车线把 / 视线拽 / 一直带 / 我回那条巷 |

## 模板 11：Hook 重复型

### 节奏感觉
简单、好记，适合 AI 演唱。

### 例句
- 今晚别 / 关灯 / 别关灯 / 别关灯 |
- 心事让 / 风吹 / 让风吹 / 让风吹 |

## 模板 12：4 bars 组合练习

### 版本 A：叙事稳拍
1. 我从那 / 条旧街 / 一路走 / 到天发亮 |
2. 风把我 / 的名字 / 吹进了 / 每扇窗框 |
3. 心里的 / 那团火 / 没熄灭 / 只是更烫 |
4. 所以我 / 还站着 / 对世界 / 把话说响 |

### 版本 B：Trap 推进
1. 别靠近 / 我状态 / 在升高 / 不会降 |
2. 这节奏 / 太滚烫 / 你跟上 / 别乱晃 |
3. 一半稳 / 一半快 / 句尾落 / 在天上 |
4. 抢拍进 / 再后放 / 让整段 / 更嚣张 |

### 版本 C：留白 + 三连音
1. 先别急 / -- / 让我把 / 气口理顺 |
2. 再推进 / -- / 把这段 / 韵脚钉稳 |
3. 别着急 / 看着我 / 跟着走 / 别退后 |
4. 最后再 / 切回稳 / 让尾句 / 落更狠 |

## 第四部分：直接可抄的填空模板

## 1. 句尾主韵模板

- 我把 ______ 写进了夜色的 ang |
- 你把 ______ 留在了背后的 ang |
- 这座城 ______ 但我没有退让的 ang |
- 风一吹 ______ 还在原地回响的 ang |

## 2. 二字组模板

- ______ / ______ / ______ / ______ |
- ______ / ______ / ______ / 韵脚 |

例如：
- 旧楼 / 冷风 / 心动 / 失重 |
- 这城 / 太硬 / 我仍 / 前进 |

## 3. Hook 模板

- 别再 ______ / 别再 ______ |
- 今晚 ______ / 让我 ______ |
- 如果 ______ / 就让 ______ |

## 4. 三连音模板

- 别着急 / 跟着拍 / 看着我 / 先呼吸 |
- 不后退 / 往前推 / 把字压 / 更整齐 |

## 第五部分：最实用练法

## 练法 1：一周只练 3 个韵脚

比如：
- ang
- an
- ing

每个韵脚各写：
- 8 个句尾字
- 4 句单押
- 4 句双押
- 1 段 4 bars

## 练法 2：同一段词换 3 种 cadence

同样一句意思：
- 稳拍写一遍
- 二连音写一遍
- 三连音写一遍

## 练法 3：每次只练一个重点

不要一口气要求自己同时做到：
- 双押
- 三连音
- 抢拍
- 留白
- hook 记忆点

最好的方法是：一首歌只主练 1-2 个技术点。

## 第六部分：最推荐优先掌握的组合

### 新手优先
- ang 主韵
- 稳拍叙事
- 一拍两字
- 句尾双押

### 进阶优先
- ing / eng 主韵
- 内部押韵
- 2 + 2 + 3 结构
- 局部三连音

### AI 生成优先
- 韵脚少换
- 每句长度接近
- hook 重复更明显
- 句尾少落虚词
