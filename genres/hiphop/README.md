# 🎤 嘻哈/说唱 (Hip-Hop)

## 核心特征

嘻哈音乐以**Beat（节拍）、Flow（说唱韵律）、采样（Sampling）、押韵歌词**为四大支柱，起源于美国黑人社区，现为全球最大流行音乐类型之一。

---

## 基础参数

| 参数 | 典型值 |
|---|---|
| BPM | 70 – 100（体感律动，通常是实际 BPM 的一半）|
| 常用调式 | 小调为主（A小调、D小调）、五声音阶 |
| 歌曲时长 | 2:30 – 4:00 |
| 人声特征 | 说唱（Rap）为主，副歌可以唱（Hook）|

---

## 歌曲结构

```
Intro
→ Verse 1 (说唱，16小节)
→ Hook / Chorus (旋律型副歌，8小节)
→ Verse 2 (说唱，16小节)
→ Hook
→ Bridge 或 Verse 3
→ Hook
→ Outro
```

---

## Beat 构成

| 元素 | 说明 |
|---|---|
| Kick | 通常在 1、3 拍或 1、2.5、3 拍（带切分）|
| Snare / Clap | 通常在 2、4 拍 |
| Hi-hat | 密集型（Trap: 32分音符）或律动型 |
| 808 Bass | Trap 的标志性元素，长滑音低音 |
| 采样 | 翻用老唱片的旋律/人声片段 |
| Melody | 合成器旋律、钢琴采样 |

---

## 子风格

| 子风格 | BPM | 特点 | 代表艺人 |
|---|---|---|---|
| Old School | 85–100 | 采样重、Funk 影响 | Nas, Jay-Z |
| Trap | 130–145（半速 65–72）| 808、三连音 hi-hat | Travis Scott, Future |
| Drill | 130–145 | 黑暗、滑音 bass、芝加哥/英国 | Pop Smoke, Central Cee |
| Boom Bap | 85–95 | 真实鼓采样，老派 | Kendrick Lamar, J. Cole |
| Lo-fi Hip-Hop | 70–90 | 慵懒、失真、学习/放松氛围 | Nujabes, Lofi Girl |
| Phonk | 130–140（半速）| 暗黑、Drift 文化、采样 | Memphis rap |
| Conscious Rap | 80–95 | 歌词深度，社会议题 | Kendrick Lamar |

## StylePrompt 推荐标签

```
hip-hop, [子风格], [BPM] BPM, [音色特征], [情绪标签],
rap vocals / hook vocals, [主题]
```

**示例（Trap）：**
```
trap, dark, 140 BPM, 808 bass, hi-hat rolls, male rap vocals,
atmospheric, drill influenced, hard-hitting, distorted bass
```

**示例（Lo-fi Hip-Hop）：**
```
lo-fi hip hop, chill, relaxing, 85 BPM, vinyl crackle, jazz samples,
mellow, study music, no lyrics, sampled piano
```
