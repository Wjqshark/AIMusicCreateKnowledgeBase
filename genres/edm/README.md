# 🎛️ 电子舞曲 (EDM — Electronic Dance Music)

## 核心特征

EDM 以**合成器音色、4/4 拍强力 Kick、Drop 结构、循环编排**为核心，为舞池设计，强调能量积累与释放。

---

## 基础参数

| 子风格 | BPM 范围 |
|---|---|
| House | 120 – 130 |
| Deep House | 118 – 124 |
| Techno | 130 – 150 |
| Trance | 128 – 145 |
| Drum & Bass | 160 – 180 |
| Dubstep | 138 – 142 |
| Future Bass | 140 – 160 |
| Trap (EDM) | 70 – 100（半速感）|

---

## 标准 EDM 结构（Drop 结构）

```
Intro (16–32 小节): 基础 beat 建立
→ Build-Up (16 小节): 能量逐渐积累（音量上升、鼓组变化、白噪声）
→ Drop (32 小节): 最大能量释放（重型 Bass、全力节拍）
→ Break (16 小节): 能量减退（旋律段、人声段）
→ Build-Up 2
→ Drop 2（通常比 Drop 1 更猛）
→ Outro (16 小节): 逐渐淡出
```

---

## 典型音色配置

| 层次 | 音色 |
|---|---|
| 节奏骨架 | Kick（4 on the floor）、Snare/Clap（2、4拍）、Hi-hat（16分）|
| 低频能量 | Sub Bass、808 Bass |
| 旋律/人声 | 合成器 Lead、人声 Hook（可 Chop）|
| 和弦层 | Synth Pad、Pluck |
| 氛围 | Riser、White Noise、FX 音效 |
| 特效 | Sweep、Crash、Impact |

---

## 子风格详解

| 子风格 | 关键特征 | 代表艺人 |
|---|---|---|
| House | 4/4 kick，灵魂/人声采样 | Disclosure, Duke Dumont |
| Deep House | 低沉、氛围感、律动慵懒 | Larry Heard, Burial |
| Progressive House | 渐进式发展，情感旋律 | Eric Prydz, deadmau5 |
| Trance | 高昂旋律，上升感，情感爆发 | Armin van Buuren, Tiësto |
| Techno | 工业感、极简、重复驱动 | Charlotte de Witte |
| Future Bass | 和弦 Chops、情感、Trap 影响 | Flume, Marshmello |
| Dubstep | Wobble Bass、猛烈 Drop | Skrillex, Excision |

## StylePrompt 推荐标签

```

[情绪标签], [人声/无人声], electronic production
```

```
future bass, uplifting, female vocal chops, supersaws, heavy drop,
emotional, 150 BPM, bright synths, festival, Flume inspired
```

```
deep house, minimal, atmospheric, 122 BPM, groovy, late night,
subtle vocals, warm bass, Chicago house feel
```
