# 🎷 爵士音乐 (Jazz)

## 核心特征

爵士以**即兴演奏、复杂和声（七和弦、九和弦、变化音）、Swing节奏感**为核心，强调乐手之间的互动与对话。

---

## 基础参数

| 参数 | 典型值 |
|---|---|
| BPM | 60（Ballad）– 300（Bebop）|
| 常用调式 | F大调、降B大调、降E大调（管乐友好）|
| 歌曲时长 | 3 – 10 分钟（含即兴段落）|
| 节奏特征 | Swing（三连音律动）或 Straight（偶数律动）|

---

## 歌曲结构

```
头 (Head): 主题旋律演奏一遍
→ 独奏 (Solos): 各乐器即兴（基于 Chorus 和弦循环）
→ 头 (Head Out): 主题旋律再现
→ 结尾 (Tag/Coda)

标准曲式: AABA (32小节) | 12小节蓝调
```

---

## 常用和弦进行

| 名称 | 和弦 | 特点 |
|---|---|---|
| 二五一 (II-V-I) | Dm7 – G7 – Cmaj7 | 爵士核心进行 |
| 蓝调 Blues | I7 – IV7 – V7（12小节）| 蓝调基础 |
| 节奏变化 Rhythm Changes | 基于 "I Got Rhythm" | Bebop 常用 |
| 转调循环 Coltrane Changes | 每4小节转调大三度 | 现代爵士 |

---

## 典型乐器配置

| 角色 | 乐器 |
|---|---|
| 旋律/独奏 | 萨克斯、小号、长号、钢琴 |
| 和弦层 | 钢琴（Comping）、吉他 |
| 低频层 | 双低音提琴（Upright Bass）|
| 节奏 | 爵士鼓（Brush 或 Stick）|
| 人声（可选）| 爵士歌手（Scat、标准曲）|

---

## 子风格

| 子风格 | BPM | 特点 |
|---|---|---|
| Swing | 120–200 | 经典爵士大乐队律动 |
| Bebop | 180–300 | 快速、复杂、1940s |
| Cool Jazz | 80–140 | 放松、抒情、Miles Davis |
| Fusion | 90–160 | 爵士+摇滚/电子 |
| Smooth Jazz | 70–110 | 商业、流畅、收音机友好 |
| Nu-Jazz | 80–120 | 现代制作+爵士元素 |
| Bossa Nova | 100–130 | 巴西节奏+爵士和声 |
