# 🎵 音阶与调式 (Scales & Modes)

## 基础音阶

### 自然大调音阶 (Major Scale)
```
公式: 全 全 半 全 全 全 半
C大调: C D E F G A B C
特点: 明亮、积极、开心
常用风格: 流行、乡村、民谣
```

### 自然小调音阶 (Natural Minor / Aeolian)
```
公式: 全 半 全 全 半 全 全
A小调: A B C D E F G A
特点: 忧郁、内省、情感深沉
常用风格: R&B、嘻哈、摇滚
```

### 和声小调 (Harmonic Minor)
```
公式: 全 半 全 全 半 增2度 半
特点: 异域感、戏剧性，V和弦有强烈导向感
常用风格: 古典、金属、弗拉门戈
```

### 旋律小调 (Melodic Minor)
```
上行: 全 半 全 全 全 全 半
下行: 同自然小调
特点: 流畅上行，爵士常用
常用风格: 爵士、现代古典
```

## 五声音阶 (Pentatonic Scales)

### 大调五声 (Major Pentatonic)
```
公式: 全 全 小三度 全 小三度
C大调五声: C D E G A
特点: 明亮、朗朗上口、无半音冲突
常用风格: 流行、民谣、乡村、中国传统音乐
即兴提示: 在大调歌曲上随意弹不会出错
```

### 小调五声 (Minor Pentatonic)
```
公式: 小三度 全 全 小三度 全
A小调五声: A C D E G
特点: 蓝调感、摇滚味
常用风格: 摇滚吉他独奏、蓝调、嘻哈
即兴提示: 摇滚吉他手最爱，在小调歌曲上随意弹
```

### 蓝调音阶 (Blues Scale)
```
A蓝调: A C D Eb E G A
（在小调五声基础上加入降五度 "蓝调音"）
特点: 独特的蓝调张力感
常用风格: 蓝调、摇滚、R&B
```

### 中国五声音阶 (Chinese Pentatonic)
```
宫调式（类大调）: C D E G A
商调式: D E G A C
角调式: E G A C D
徵调式: G A C D E
羽调式（类小调）: A C D E G
特点: 东方色彩、无半音
常用风格: 中国流行、民谣、游戏音乐
```

## 调式 (Modes)

以C大调各音为起点衍生出的7个调式：

| 调式 | 起始音 | 与大调关系 | 情绪特征 | 常用风格 |
|---|---|---|---|---|
| Ionian（大调）| C | = C大调 | 明亮、正常 | 流行、民谣 |
| Dorian | D | D为根音 | 小调但稍明亮 | 爵士、R&B、流行 |
| Phrygian | E | E为根音 | 暗沉、西班牙感 | 金属、弗拉门戈 |
| Lydian | F | F为根音 | 梦幻、上扬感 | 电影配乐、爵士 |
| Mixolydian | G | G为根音 | 大调但降七度，蓝调感 | 摇滚、蓝调、凯尔特 |
| Aeolian（小调）| A | = A自然小调 | 忧郁、内省 | 通用小调 |
| Locrian | B | B为根音 | 极度不稳定 | 极少使用 |

## 常用调性推荐

| 场景/需求 | 推荐调 | 原因 |
|---|---|---|
| 吉他弹唱，开放弦 | G、D、A、E | 吉他最自然 |
| 钢琴，简洁 | C | 全白键 |
| 管乐器友好 | F、降B、降E | 移调乐器传统调 |
| 女声流行 | A小调、C大调、G大调 | 常见音域 |
| 男声流行 | E小调、G大调、D大调 | 常见音域 |
| 中国/东方风 | D宫调式、G宫调式 | 五声音阶友好 |
| 哀伤、悲情 | D小调、A小调 | 传统"悲调" |
| 明亮欢快 | C大调、G大调 | 干净无升降号 |
